Types

A type tells whitelister what to allow through its filter (or validate against). Different types support varying rules for whitelister to validate against.

Types can be passed as a rule: {% highlight js %} const rules = { user_id: { type: 'integer' }, }; {% endhighlight %} Or shorthand when alone: {% highlight js %} const rules = { user_id: 'integer', }; {% endhighlight %} Currently, whitelister doesn't support "custom" types, but it is being discussed.
